# Blockchain-Based Digital Identity System

## Description
This project is a digital identity system using Python Flask for backend and HTML/JS frontend. It simulates storing and verifying identity hashes, inspired by blockchain principles.

## Features
- Create digital identity (hash)
- Verify digital identity
- Simulated blockchain transaction
- Simple web interface

## How to Run
1. Run backend: `python app.py`
2. Open `index.html` in browser
3. Enter name and email → click "Create Identity" or "Verify Identity"
